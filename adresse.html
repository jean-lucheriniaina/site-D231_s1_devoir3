Informations de Déploiement

Le site a été déployé avec succès en ligne sur l'adresse suivante :

Pousser projet sur GitHub :

git init
git add .
git commit -m "Déploiement initial"
git branch -M main
git remote add origin https://github.com/{utilisateur}/{nom-de-repo}.git
git push -u origin main

Activer GitHub Pages :

  • Dans Settings → Pages
  • Sélectionne la branche main ou gh-pages et le dossier / ou /docs
  • Le site sera disponible sous https://utilisateur.github.io/mon-repo/
